# Caipiao Predictor macOS App

这是一个原生 SwiftUI macOS 应用，当前实现福彩/体彩主流全国数字彩工作台。

## 功能

- 启动时载入所有支持彩种的内置历史开奖数据
- 支持从中彩网 JSONP 接口更新当前彩种或全部彩种
- 支持福彩：快乐8、双色球、福彩3D、七乐彩
- 支持体彩：超级大乐透、排列3、排列5、七星彩
- 支持选择玩法，例如快乐8选一至选十、双色球标准/复式参考、3D直选/组三/组六、大乐透基本/追加等
- 支持选择并查看任意已加载期号
- 点击“预测下一期”后，按所选彩种和玩法输出候选号码
- 支持选择综合统计、热度趋势、遗漏回补、机器学习、神经网络等算法
- 支持查看当前算法说明
- 展示模型评分 Top 30，方便查看排序依据

## 运行


## 模型说明

球类彩种按号码分区建模，例如双色球红球/蓝球、大乐透前区/后区。数字位彩种按位置建模，例如 3D 的百十个位。预测器使用四类历史特征：

- 全历史出现频率
- 近 100 期出现频率
- 近 30 期出现频率
- 当前遗漏期数

模型输出只代表历史数据下的候选排序。彩票开奖应视为独立随机事件，应用不会也不能保证中奖。

机器学习算法使用本地轻量逻辑回归训练历史滚动样本，神经网络算法使用单隐藏层前馈网络训练同类滚动样本，二者都会输出概率评分；统计类算法会在每次更新最新开奖后重新计算特征和模型评分。
