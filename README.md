# Lottery Ticket Prediction

一个原生 SwiftUI macOS 彩票数据工作台，支持多种彩票的历史开奖查看、统计建模、候选号码生成、中奖规则查询和指定期号兑奖测算。

> 重要提醒：彩票开奖结果应视为独立随机事件。本项目用于数据分析、建模练习和规则查询，不构成中奖承诺，也不建议超预算投注。

## 支持彩种

| 类型 | 彩种 |
| --- | --- |
| 中国福利彩票 | 快乐8、双色球、福彩3D、七乐彩 |
| 中国体育彩票 | 超级大乐透、排列3、排列5、七星彩 |

## 核心功能

- 启动时载入内置历史开奖数据
- 支持更新当前彩种或全部彩种
- 支持按彩种切换玩法、算法和历史期号
- 支持多算法预测下一期候选号码
- 支持综合色彩显示红球、蓝球、前区、后区、数字位等不同分区
- 支持算法对比和综合推荐号码
- 支持查看号码评分榜
- 支持查看中奖规则
- 支持输入指定期号和彩票号码，测算奖级、测算注数、固定奖金和浮动奖提示
- 支持快乐8、双色球、大乐透、七乐彩等复式候选的拆注测算

## 预测算法

当前 App 内置以下候选生成方式：

- 综合统计：结合长期频率、近100期、近30期和遗漏
- 热度趋势：更偏向近期活跃号码
- 遗漏回补：提高久未出现号码的观察权重
- 机器学习：用历史滚动样本训练轻量逻辑回归
- 神经网络：用单隐藏层前馈网络学习非线性组合
- 冷门组合：避开常见投注偏好，偏向更分散的候选形态

球类彩种按分区建模，例如双色球红球/蓝球、大乐透前区/后区；数字位彩种按位置建模，例如福彩3D、排列3、排列5、七星彩。

## 项目结构

```text
.
├── CaipiaoPredictor/        # SwiftUI macOS 应用源码和 SwiftPM 配置
├── scripts/                 # 数据抓取、内置数据生成和分析脚本
├── data/                    # 历史开奖数据
├── outputs/                 # 分析输出
├── README.md                # 项目说明
└── README.zh-CN.md          # 中文说明
```

## 运行

```bash
cd CaipiaoPredictor
swift run
```

## 打包为 macOS App

```bash
cd CaipiaoPredictor
Scripts/package_app.sh
open ../build/Caipiao\ Predictor.app
```

打包产物会生成在根目录 `build/` 下，该目录不会提交到 Git。

## 数据脚本

更新 App 内置开奖数据：

```bash
python3 scripts/fetch_bundled_draws.py
```

快乐8专项抓取和分析脚本：

```bash
python3 scripts/kl8/scrape.py
python3 scripts/kl8/analyze.py
```

## 数据来源

- 中彩网前端 JSONP 接口
- 广东省福利彩票发行中心历史页备份
- 中国福利彩票、中国体育彩票公开玩法规则

## 构建验证

```bash
cd CaipiaoPredictor
swift build
```

当前 macOS App 目标平台为 macOS 14 及以上。

## 免责声明

本项目不会也不能改变彩票开奖概率。机器学习、神经网络和统计模型输出只代表历史数据下的候选排序，不代表未来开奖优势。兑奖测算仅用于规则核对，最终以官方开奖公告和中奖彩票为准。
